Odoo - Deployment

Placeholder

PROJECT DIRECTORY

Direktori pada server yang menyimpan docker compose untuk project tertentu. Nama direktori ini akan sesuai dengan nama <<PROJECT REPO>>. Direktori ini dibuat pada direktori home <<USER OS>>

PROJECT REPO

Nama repositori di github yang digunakan untuk menyimpan docker compose untuk project tertentu. PROJECT REPO sama dengan <<DOMAIN PRODUKSI>> yang digunakan dengan tanda titik (.) diubah menjadi dash (-).

Contoh:

  • simetri-sinergi.id menjadi simetri-sinergi-id
  • demo-kap.simetri-sinergi.id menjadi demo-kap-simetri-sinergi-id

URL PROJECT REPO

URL dari <<PROJECT REPO>>

USER OS

Nama user yang digunakan untuk SSH ke server. USER OS tidak boleh menggunakan root

DOMAIN PRODUKSI

Domain yang digunakan untuk mengakses Odoo

Prasyarat

Update dan Upgrade Repository

1. Update dan upgrade repository

Jalankan sintaks berikut:

sudo apt-get update && sudo apt-get -y upgrade

Instalasi Docker

1. Install docker & docker.io

Jalankan sintaks berikut:

sudo apt-get install docker docker.io

Instalasi Docker Compose

1. Download docker-compose terbaru

Jalankan sintaks berikut:

sudo curl -L "https://github.com/docker/compose/releases/download/1.26.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

2. Buat docker-compose menjadi executable

Jalankan sintaks berikut:

sudo chmod +x /usr/local/bin/docker-compose

Instalasi pip

1. Install pip untuk python3

Jalankan sintaks berikut:

sudo apt-get install python3-pip

Instalasi virtualenv

1. Install virtualenv untuk python3

Jalankan sintaks berikut:

sudo apt-get install python3-venv

Instalasi pipx

1. Instalasi pipx dengan menggunakan pip

Jalankan sintaks berikut:

python3 -m pip install --user pipx

2. Jalankan pipx ensurepath

Jalankan sintaks berikut:

python3 -m pipx ensurepath

Instalasi copier

1. Instalasi copier dengan menggunakan pipx

Jalankan sintaks berikut:

pipx install copier

Instalasi invoke

1. Instalasi invoke dengan menggunakan pipx

Jalankan sintaks berikut:

pipx install invoke

Instalasi pre-commit

1. Instalasi precommit dengan menggunakan pipx

Jalankan sintaks berikut:

pipx install pre-commit

Buat SSH Key

1. Masuk ke direktori home

Jalankan sintaks berikut:

cd ~

2. Buat SSH Key

Jalankan sintaks berikut:

ssh-keygen -t rsa -b 4096

Inverseproxy

Instalasi

1. Pastikan working directory berada pada home directory user OS

Jalankan sintaks berikut:

cd ~

Abaikan langkah ini jika Anda sudah berada pada home directory

2. Clone aplikasi inverseproxy

Jalankan sintaks berikut:

git clone https://github.com/open-synergy/docker-compose-inverseproxy.git inverseproxy

Keterangan placeholder:

  • {nama-folder-inverse-proxy}. Nama folder yang akan dibuat untuk menyimpan kode inverseproxy. Nama folder bebas.

3. Masuk ke direktori inverseproxy

Jalankan sintaks berikut:

cd inverseproxy

Keterangan placeholder:

  • inverseproxy. Nama folder tempat menyimpan kode inverseproxy. Harus sesuai dengan nama folder pada langkah ke-2

4. Sesuaikan nilai ACME.Email pada file inverseproxy.yaml

Ubah nilai ACME.Email pada file inverseproxy.yaml menjadi alamat email yang valid.

Menjalankan Aplikasi

1. Masuk ke direktori

Jalankan sintaks berikut:

cd {nama-folder-inverse-proxy}

Keterangan placeholder:

  • {nama-folder-inverse-proxy}. Nama folder tempat menyimpan kode inverseproxy. Harus sesuai dengan nama folder pada langkah ke-2 halaman instalasi

Abaikan langkah ini jika Anda sudah berada pada {nama-folder-inverse-proxy}

2. Jalankan aplikasi

Jalankan sintaks berikut:

sudo docker-compose -f inverseproxy.yaml up -d
