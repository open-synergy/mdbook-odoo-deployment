Odoo - Deployment

Prasyarat

Update dan Upgrade Repository

1. Update dan upgrade repository

Jalankan sintaks berikut:

sudo apt-get update && sudo apt-get -y upgrade

Instalasi Docker

1. Install docker & docker.io

Jalankan sintaks berikut:

sudo apt-get install docker docker.io

Instalasi Docker Compose

1. Download docker-compose terbaru

Jalankan sintaks berikut:

sudo curl -L "https://github.com/docker/compose/releases/download/1.26.0/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose

2. Buat docker-compose menjadi executable

Jalankan sintaks berikut:

sudo chmod +x /usr/local/bin/docker-compose

Instalasi pip

1. Install pip untuk python3

Jalankan sintaks berikut:

sudo apt-get install python3-pip

Instalasi virtualenv

1. Install virtualenv untuk python3

Jalankan sintaks berikut:

sudo apt-get install python3-venv

Instalasi pipx

1. Instalasi pipx dengan menggunakan pip

Jalankan sintaks berikut:

python3 -m pip install --user pipx

2. Jalankan pipx ensurepath

Jalankan sintaks berikut:

python3 -m pipx ensurepath

Instalasi copier

1. Instalasi copier dengan menggunakan pipx

Jalankan sintaks berikut:

pipx install copier

Instalasi invoke

1. Instalasi invoke dengan menggunakan pipx

Jalankan sintaks berikut:

pipx install invoke

Instalasi pre-commit

1. Instalasi precommit dengan menggunakan pipx

Jalankan sintaks berikut:

pipx install pre-commit

Buat SSH Key

1. Masuk ke direktori home

Jalankan sintaks berikut:

cd ~

2. Buat SSH Key

Jalankan sintaks berikut:

ssh-keygen -t rsa -b 4096

Inverseproxy

Instalasi

1. Pastikan working directory berada pada home directory user OS

Jalankan sintaks berikut:

cd ~

Abaikan langkah ini jika Anda sudah berada pada home directory

2. Clone aplikasi inverseproxy

Jalankan sintaks berikut:

git clone https://github.com/open-synergy/docker-compose-inverseproxy.git {nama-folder-inverse-proxy}

Keterangan placeholder:

  • {nama-folder-inverse-proxy}. Nama folder yang akan dibuat untuk menyimpan kode inverseproxy. Nama folder bebas.

3. Masuk ke direktori {nama-folder-inverse-proxy}

Jalankan sintaks berikut:

cd {nama-folder-inverse-proxy}

Keterangan placeholder:

  • {nama-folder-inverse-proxy}. Nama folder tempat menyimpan kode inverseproxy. Harus sesuai dengan nama folder pada langkah ke-2

4. Sesuaikan nilai ACME.Email pada file inverseproxy.yaml

Ubah nilai ACME.Email pada file inverseproxy.yaml menjadi alamat email yang valid.

Menjalankan Aplikasi

1. Masuk ke direktori

Jalankan sintaks berikut:

cd {nama-folder-inverse-proxy}

Keterangan placeholder:

  • {nama-folder-inverse-proxy}. Nama folder tempat menyimpan kode inverseproxy. Harus sesuai dengan nama folder pada langkah ke-2 halaman instalasi

Abaikan langkah ini jika Anda sudah berada pada {nama-folder-inverse-proxy}

2. Jalankan aplikasi

Jalankan sintaks berikut:

sudo docker-compose -f inverseproxy.yaml up -d

Odoo

Instalasi

1. Pastikan working directory berada pada home directory user OS

Jalankan sintaks berikut:

cd ~

Abaikan langkah ini jika Anda sudah berada pada home directory

2. Buat direktori untuk meletakan aplikasi

Jalankan sintaks berikut:

mkdir {nama-folder-odoo}

Keterangan placeholder:

  • {nama-folder-odoo}. Penamaan {nama-folder-odoo} disamakan dengan domain production yang akan digunakan. Apabila domain production yang akan digunakan adalah latihan.simetri-sinergi.id, maka {nama-folder-odoo} adalah latihan1.simetri-sinergi.id

3. Copy aplikasi dengan menggunakan copier

Jalankan sintaks berikut:

copier copy {alamat-copier-template} {nama-folder-odoo}

Keterangan placeholder:

  • {alamat-copier-template}. URL copier template yang digunakan. Daftar copier template yang dapat digunakan dapat dilihat disini.
  • {nama-folder-odoo}. Nama folder sesuai dengan {nama-folder-odoo} pada langkah ke-2

Jawab pertanyaan-pertanyaan yang muncul. Pertanyaan-pertanyaan yang diajukan tergantung dari template copier yang digunakan.

4. Masuk ke direktori {nama-folder-odoo}/odoo/custom/ssh

Jalankan sintaks berikut:

cd {nama-folder-odoo}/odoo/custom/ssh

Keterangan placeholder:

  • {nama-folder-odoo}. Nama folder sesuai dengan {nama-folder-odoo} pada langkah ke-2

5. Copy isi ssh private key ke dalam file id_rsa

Jalankan sintaks berikut

cat {path-to-ssh-private-key} > id_rsa

Keterangan placeholder:

  • {path-to-ssh-private-key}. Path ke file ssh private key yang dibuat sesuai dengan instruksi kerja Membuat SSH Key

6. Copy isi ssh public key ke dalam file id_rsa

Jalankan sintaks berikut

cat {path-to-ssh-public-key} > id_rsa.pub

Keterangan placeholder:

  • {path-to-ssh-public-key}. Path ke file ssh public key yang dibuat sesuai dengan instruksi kerja Membuat SSH Key

7. Build aplikasi

Jalankan sintaks berikut:

sudo docker-compose -f prod.yaml build --no-cache

Menjalankan Aplikasi

1. Masuk ke direktori {nama-folder-odoo}

Jalankan sintaks berikut:

cd {nama-folder-odoo}

Keterangan placeholder:

  • {nama-folder-odoo}. Nama folder sesuai dengan {nama-folder-odoo} pada langkah ke-2 halaman instalasi

Abaikan langkah ini jika ada sudah berada pada {nama-folder-odoo}

2. Jalankan aplikasi

Jalankan sintaks berikut:

sudo docker-compose -f prod.yaml up -d

Menghentikan Aplikasi

1. Masuk ke direktori {nama-folder-odoo}

Jalankan sintaks berikut:

cd {nama-folder-odoo}

Keterangan placeholder:

  • {nama-folder-odoo}. Nama folder sesuai dengan {nama-folder-odoo} pada langkah ke-2 halaman instalasi

Abaikan langkah ini jika ada sudah berada pada {nama-folder-odoo}

2. Menghentikan aplikasi

Jalankan sintaks berikut:

sudo docker-compose -f prod.yaml down

Instalasi Addons Odoo

1. Masuk ke direktori {nama-folder-odoo}

Jalankan sintaks berikut:

cd {nama-folder-odoo}

Keterangan placeholder:

  • {nama-folder-odoo}. Nama folder sesuai dengan {nama-folder-odoo} pada langkah ke-2 halaman instalasi

Abaikan langkah ini jika ada sudah berada pada {nama-folder-odoo}

2. Hentikan service odoo

Jalankan sintaks berikut:

sudo docker-compose -f prod.yaml stop odoo

3. Tentukan modul-modul yang akan diinstall

Jalankan sintaks berikut untuk menentukan modul yang diinstall secara manual:

modules=modul-1,modul-2,modul-3,...,modul-n

atau

Jalankan sintaks berikut untuk menginstall modul-modul yang telah didefinisikan dalam file tertentu

modules=$(<{path-to-file})

Keterangan placeholder:

  • {path-to-file}. Path ke file yang berisi daftar modul yang akan diinstall.

4. Instalasi modul-modul Odoo

Jalankan sintaks berikut:

sudo docker-compose -f prod.yaml run --rm odoo odoo -i $modules --without-demo --stop-after-init

5. Jalankan service odoo

Jalankan sintaks berikut:

sudo docker-compose -f prod.yaml start odoo
